# Actividad_Final_UD1_Economia_Circular_exeLearning_SCP
Como parte de un equipo SMR contratado por una pequeña empresa industrial o de servicios, desarrollas un recurso digital para formar al personal en Economía Circular aplicada a las TIC: uso eficiente de la tecnología, reducción de residuos, reutilización de equipos y alineación con los ODS.
